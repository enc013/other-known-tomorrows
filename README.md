# Other Known Tomorrow Project

**AUTHOR**: Enrique Carrillosulub (Institute of Computational Neuroscience, UCSD)
**PI**: Ying Choon Wu (Institute of Computational Neuroscience, UCSD) & Rhonda Holberton (College of Humanities and the Arts, San Jose State University)

# Description
This repository stores all preprocessing and feature extraction for Rhonda Holberton's work "Other Known Tomorrows" in collaboration with Ying Choon Wu. 
Our work transforms brain-wave activity that was collected from human participants responding to AI-generated future spectuations into AI-manipulated video.
EEG data is not included.
